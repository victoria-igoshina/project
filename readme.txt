Ограничения для сортировки:
Файл из символов ASCII
В файле не больше 2^63 символов (~16000Pb)
Наличие папки tmp, принимаемой программой
Наличие свободного места на жестком диске 2*(размер input файла)

Генератор генерит от 32-го до 126-го символы ASCII

В run_test.sh есть пример сборки и запуска бинарников генератора файла и сортировки файла.
